# JOB-API

API for JOB Application Management System



## Installation

Clone the repository

Install the dependencies

Fill the necessary environment variables (see .env.example)

Run the server

```bash
git clone git@github.com:Ryrahul/Job-Api.git
cd JOB-API
npm
npm run dev
```
